# v6.3.4 Position Close & Cleanup API + Test

## Version

v6.3.4-position-close-cleanup-test

## Goal

Add and validate maintenance APIs for closing an existing demo-fapi position and cleaning up residual regular orders and algo/conditional protection orders.

## Code Changes

Added maintenance APIs:

- POST /positions/{symbol}/close
- POST /positions/{symbol}/cleanup

The close API:

- Requires X-Runtime-Control-Token
- Works while Runtime is locked
- Does not unlock Runtime
- Rejects non-demo Binance environments when live-endpoint rejection is enabled
- Cancels existing regular and algo orders before close when requested
- Closes the current position using reduceOnly MARKET order
- Polls for flat position
- Cancels residual regular and algo orders after close when requested

The cleanup API:

- Requires X-Runtime-Control-Token
- Works while Runtime is locked
- Does not close positions
- Cancels residual regular and algo/conditional orders for the symbol
